# DiskLens Downloads

This folder hosts the installer files for DiskLens.

## Naming Convention

```
DiskLens-{version}-{platform}.{ext}
```

### Examples:
- `DiskLens-1.0.18-mac-arm64.dmg` - macOS Apple Silicon
- `DiskLens-1.0.18-mac-x64.dmg` - macOS Intel
- `DiskLens-1.0.18-win-x64.exe` - Windows 64-bit
- `DiskLens-1.0.18-win-x64-setup.exe` - Windows installer
- `DiskLens-1.0.18-win-x64-portable.zip` - Windows portable

For security verification, SHA-256 checksums are provided in `checksums.txt`.

## System Requirements

### macOS
- macOS 11 (Big Sur) or later
- Apple Silicon or Intel processor

### Windows
- Windows 10 or Windows 11
- 64-bit processor
